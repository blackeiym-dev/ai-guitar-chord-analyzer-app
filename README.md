# Guitar Analyzer

ブラウザだけで動作するギター練習・コード進行解析アプリです。

## 主な機能
- YouTube埋め込み再生、A-Bループ、速度変更、区間保存
- MP3/WAV/M4A/動画のローカル再生、波形、A-Bループ
- ローカル音源のコード進行自動解析（キー、BPM、小節コード候補）
- 解析コードの小節ごとの手動修正
- 修正後コード進行を練習画面へ反映
- 現在コード・次コード表示、コード位置へのジャンプ
- コード進行とコードトーン指板の自動連動
- スケール、トライアド、CAGED、メトロノーム
- ブラウザ保存、JSON入出力、印刷/PDF

## 重要
YouTube音声は解析しません。コード自動解析は、利用者が手元の音源ファイルを選択した場合に端末内で実行します。解析結果は候補として表示され、手動で修正できます。
